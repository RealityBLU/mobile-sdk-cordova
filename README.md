# Setup plugin

## iOS
    1 - `cordova plugin add `
    2 - `cordova platform add ios`
    3 - sign code
    4 - `cordova build ios`

    Note: native sdk works only with real devices (not simulators)
## Android
    1 - `cordova plugin add `
    2 - `cordova platform add android`
    3 - `cordova build android`
